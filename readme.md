# Finance Sales Performance Dashboard (Power BI)

## Project Overview

This Power BI dashboard provides insights into sales performance, target achievement, and variance trends, enabling data-driven decision-making through two years of sales data.

## Dataset Description

The dataset includes actual sales, sales targets, calendar data, and salesperson information for 25 sales representatives from January 2023 to February 2024 which is uploaded in the data folder. It was used to analyze sales performance, target achievement, variance trends, and team-level performance in Power BI.

## Business Problem

Organizations often struggle to quickly identify:

* Whether sales targets are being achieved
* Which sales representatives are driving performance
* Trends impacting revenue growth or decline
* Areas requiring corrective action

This dashboard addresses these challenges by providing an interactive and centralized view of sales performance metrics.

## Dashboard Features

### Key Performance Indicators (KPIs)

* Actual Sales
* Target Sales
* Variance Analysis
* Sales Achievement Percentage

### Visualizations

* Actual vs Target Sales Comparison
* Monthly Sales Trend Analysis
* Sales Variance Trend
* Salesperson Performance Breakdown
* Team-Level Performance Metrics

### Interactive Functionality

* Dynamic Filters and Slicers
* Drill-Down Analysis
* Cross-Filtering Across Visuals
* Year and Month-Based Analysis

## Key Insights Generated

* Identified periods of sales growth and decline.
* Highlighted top-performing and underperforming sales representatives.
* Measured gaps between actual and target sales.
* Enabled management to track sales performance trends over time.
* Supported data-driven decision-making through interactive reporting.
